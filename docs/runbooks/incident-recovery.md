# Incident Recovery

## If Telegram entry fails
- verify bot token and adapter
- verify bot can receive /start
- restart session binding service

## If local standard execution fails
- verify System A standard profile
- offer session reset
- offer retry

## If local large execution fails
- verify large profile availability
- offer fallback to Terminal Agent
- explain limitation clearly

## If offload execution fails
- verify System B reachability
- verify offload contract health
- offer fallback scenario
