# OpenClaw Agentic Intel Web Demo

Static web demo surface for showing:

- OpenClaw orchestration
- LiteLLM model routing
- SambaNova inference
- System A (CWF) and System B (GNR) execution paths
- demo console explaining what answers where and why

## Run locally

```bash
python3 -m http.server 8080
```

Then open:

```text
http://localhost:8080
```

## Public demo

This folder can be served as static files from any HTTP server.
