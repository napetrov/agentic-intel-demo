Agent command
Send a free-form command to the agent gateway and watch it pick a tool
Submits task_type=agent_invoke through /api/offload; the agent gateway classifies the input (shell / read_file / list_files / summarize / echo) and runs it. read and list are scoped to the read-only demo-workspace/ mount (files: AGENTS.md, BOOTSTRAP.md, IDENTITY.md, MEMORY.md, SOUL.md, USER.md). Try whoami, uname -a, list ., read AGENTS.md, or summarize OpenClaw orchestrates agentic execution on Intel CPUs.
